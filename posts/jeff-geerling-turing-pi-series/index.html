Turing Pi Kubernetes Cluster YouTube Series

While working on building my Raspberry Pi 4 Kubernetes cluster, I came across this excellent YouTube series from Jeff Geerling on building a 7-node Kubernetes cluster using a Turing Pi and Raspberry Pi Compute Module 3+ boards.

The series is interesting even if you already know a bit about Kubernetes, particularly episode 3 (cluster setup) and episode 4 (deploying applications).
